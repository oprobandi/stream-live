# ADR-009 — V1.3 Polish Sprint (No-Backend Improvements)

**Date:** 2026-03-23  
**Status:** Accepted

---

## Context

V1.2 resolved all deployment issues. V1.3 is a polish sprint addressing the full 🟡 (Important) backlog from TODO.md without introducing a backend or payments. All changes remain within the static, no-server constraint.

## Decisions

### 1. Amber-gold primary colour (`#fbbf24`)

The electric mint green (`#00e5a0`) was replaced with warm amber-gold (`#fbbf24`). Rationale: the existing premium indicator is red-orange (`#ff7043`); gold reads as warm and elevated — appropriate for a premium-adjacent video platform — while being visually distinct from the premium colour. All CSS variables (`--primary`, `--primary-dim`, `--primary-glow`) and hardcoded RGBA values in the hero gradient updated accordingly.

### 2. Skeleton loaders (`views.renderSkeletons()`)

Added shimmer-animated placeholder cards that appear immediately on page load, while `SL.idb.loadAll()` fetches persisted uploads asynchronously. Eliminates the flash of an empty grid. Implemented via a CSS `@keyframes shimmer` animation on gradient backgrounds.

### 3. IndexedDB persistence for uploads (`idb.js`)

Uploaded videos previously existed only in memory and were lost on page reload. A new `SL.idb` module (IndexedDB, DB name `stream-live`, version 1) stores:
- Video metadata in the `uploads` object store (blob URLs stripped)
- The raw `File` blobs in the `blobs` object store

On load, blob URLs are recreated from stored data. Failure is non-fatal — the current session always works regardless.

### 4. Mobile search bar (expandable toggle)

The search bar was hidden on <640px viewports with no alternative. A `.nav-search-toggle` icon button is now shown on mobile. Tapping it expands the search bar as a fixed panel below the navbar (class `.mobile-open`). Collapsing clears the search query. Escape key also collapses.

### 5. Keyboard focus trap in modals (WCAG 2.1 §2.1.2)

A shared `SL.trap` utility engages on every modal open and releases on close. It collects all focusable elements within the modal and intercepts `Tab`/`Shift+Tab` to cycle focus within the boundary, preventing keyboard users from reaching background content.

### 6. `<video>` poster attribute

`player.js` now sets `videoEl.poster = video.thumb` before loading the source. The thumbnail is displayed in the video area while the file loads or buffers, eliminating the black-frame flash.

### 7. Scroll position memory

`SL.scroll.lock()` saves `window.scrollY` before opening any modal. `SL.scroll.unlock()` restores it on close, preventing the page from jumping to the top after dismissing a modal. Replaces all direct `document.body.style.overflow` assignments.
